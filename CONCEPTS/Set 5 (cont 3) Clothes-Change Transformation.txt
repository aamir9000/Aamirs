# CONCEPT 83 — SNAP-CHANGE COUTURE / FOUR-LOOK SPIN-CHANGE POP RUNWAY (SILENT)

Title: Pillar 2 — Editorial Bold

Format: Fashion Film / Outfit-Change Hook Reel

Hero Product: the "Prism" mirrored sculptural clutch — a faceted chrome mini-clutch that stays in her hand through every change as the brand anchor, catching and throwing each look's pop colour (persistent continuity hero)

Setting: a high-gloss pop runway studio — a colour-shifting seamless that re-gels to match each look (tangerine → cyan → ultraviolet → gold), a high-gloss mirror floor, clean and saturated

Wardrobe (4 LOOKS): LOOK A "Voltage Tangerine" — an oversized tangerine power-blazer-dress colour-blocked with hot-pink lapels; LOOK B "Liquid Cyan" — a fluid lime-cyan liquid-sequin slip gown; LOOK C "Ultraviolet Latex" — an ultraviolet-and-magenta high-gloss latex mini with sculptural shoulders; LOOK D "Molten Gold" — a liquid-gold metallic draped column finale gown

Footwear (4): hot-pink mirror-PVC pointed pumps (Look A); clear acrylic heels with glowing lime cores (Look B); magenta high-gloss thigh-high boots (Look C); gold strappy sky-high sandals (Look D)

Accessories (4 + constant hero): the Prism chrome clutch held throughout; oversized hot-pink resin hoops + chrome chain belt (Look A); cyan crystal drop earrings + layered chrome chokers (Look B); chunky ultraviolet acrylic bangles + chrome ear-cuffs (Look C); sculptural gold collar + gold drop earrings (Look D)

Hair: — a glossy chrome-shine base restyled per look (high pony for A, sleek blunt centre-part for B, slicked wet bob-illusion for C, soft sculpted finger-wave for D); colour and face locked to reference every frame

Makeup: — graphic pop base restyled per look (tangerine gloss + pink liner A; cyan-lime graphic B; ultraviolet ombré lip C; gold-foil glow D); glass skin throughout

Eye Colour: locked to the Aira reference — warm hazel-brown with a soft amber inner ring; zero drift, both irises matched, no heterochromia

Movement: rapid on-beat spin / flip / bloom outfit changes, with one macro rack-focus → 3D orbit on the levitating Prism clutch (beauty mandate adapted, X.15 rule 16)

Concept Line (NOT spoken): "blink — and she's someone new"

Duration: 30 seconds | 8 frames

Mood: KINETIC, HIGH-FASHION, POP-MAXIMAL

Composition: centered hero with clean negative space tuned for each colour-world

Grade: vibrant maximal pop — saturated tangerine / cyan / ultraviolet / gold held crisp and clean per look, never muddy (override)

Spoken line (AR): NONE — fully silent reel, no voice-over any frame (X.15)

Audio note: diegetic sound design only per frame (no music in-clip); a separate subtle Master Track Brief sits underneath, never full

Identity safety: Aira face/bone-structure/eye colour locked to reference every frame; adult, non-sexualized; zero drift through every wardrobe change (keyframe-lock) — only clothing, footwear, accessories, hair-styling and makeup-accent change, never the face

Visual-lead note: every change must read as a clean, magical, high-fashion swap — crisp colour separation and believable physics, each look vibrant and pop, never muddy, murky or dull

Hook strategy: open on a pattern-interrupt tease (a held freeze + a snap), then escalate with on-beat changes (spin → flip-board → light-bloom), each swap a fresh visual surprise, building to a gold finale and a seamless loop back to the opening so the reel re-watches itself
