# AIRA — 5 PHOTO CAROUSELS · SWIPEABLE SETS (Heavy-Detail)

<aside>
Five carousel sets — the swipeable "photo-dump" format. Each carousel is a COHESIVE set of 6–8 stills that share one aesthetic (palette, wardrobe thread, hair, location) and tell a VIBE, not a transformation story. Slides relate by mood, not by world-state matching — no chain, no handoff. Each set gives: the cohesive look, a per-slide brief (angle/shot-size, action, key detail, palette note), a cover-slide hook, and a caption. 9:16 or 4:5. VIVID colour always. Aira adult, modest, identity-locked — likeness from reference only ("Aira, [placeholder]").
</aside>

## Global Spec
- Identity: AIRA reference-locked, adult, modest, non-sexualized, no physical enumeration. Subject intro per slide: "Aira, [placeholder]".
- Cohesion law: within a carousel, keep ONE wardrobe thread, ONE hairstyle, ONE location/palette so the set reads as one shoot. Vary only angle, shot-size, action and crop slide-to-slide.
- Distinctness law: the 5 carousels are deliberately different from each other (and from the Hero Stills) in palette, location, wardrobe and hair.
- Anti-flip: ORIENTATION LOCK on every slide — preserve the exact left-to-right composition; do not mirror, flip or invert.
- Colour law: VIVID, saturated, rich blacks, zero murk; one bold colour anchor per set.
- Detail note: each slide brief is lighter than a full hero still but still specifies angle, shot-size, action, the hero detail, light and palette so each frame is buildable.
